# Skill: Step 4 Story Alchemist Draft Writer

## 1. Description
기획된 시놉시스와 현장 분위기, 내면 묘사를 바탕으로 몰입감 넘치는 스토리 초안을 작성합니다.

## 2. Usage
```bash
python logic.py --api_key "GEMINI_API_KEY" --synopsis "시놉시스" --character "캐릭터정보" --episode "회차제목" --scene "현장묘사" --inner "속마음묘사"
```

## 3. Output
- Markdown 형식의 고몰입 블로그 초안 원고
