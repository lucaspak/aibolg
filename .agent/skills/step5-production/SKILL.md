# Skill: Step 5 Marketing Captain Production Engine

## 1. Description
전체 기획 단계를 통합하여 최종 블로그 원고를 작성하고, 각 에피소드에 어울리는 극사실주의 3D 이미지를 생성합니다.

## 2. Usage
```bash
python logic.py --api_key "GEMINI_API_KEY" --data_json "기획데이터JSON" --part "1-4"
```

## 3. Output
- JSON 형식: `{"content": "블로그원고", "images": [{"prompt": "프롬프트", "path": "이미지경로"}]}`
