# Skill: Step 3 Drama Strategy Planner

## 1. Description
캐릭터와 고객 정보를 바탕으로 러셀 브런슨의 비누극(Soap Opera) 시퀀스를 적용한 4부작 마케팅 시놉시스를 기획합니다.

## 2. Usage
```bash
python logic.py --api_key "GEMINI_API_KEY" --customer "고객정보" --character "캐릭터정보" --secret "비밀/기회" --wall "좌절" --epiphany "깨달음" --cta "제안" --strategy "Soap|Standard"
```

## 3. Output
- Markdown 형식의 4부작 시리즈 기획표 (네이버 SEO 최적화)
